# discord (15 pts)

**Category:** welcome

**Author:** LA CTF Organizers

## Description
Discord is our primary platform of communication, support, and chatter during the competition. This is how we will let you know if anything happens that needs to be said, such as challenges being updated, and is how you can contact us with issues, like if you think you have the right flag but it isn't working, or if you think one of the challenges is broken.

Join our [Discord server](https://lac.tf/discord) and see the message in #support for the flag!
