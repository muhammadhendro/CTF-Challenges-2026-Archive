# Treasure Hunt 3 - pCapsized (86 pts)

**Category:** forensics

## Description
We took a road trip to Crenshaw, asked around, and the locals immediately knew we were talking about the 'water guy'. Apparently he's been in big trouble with utility companies for regularly not paying his water bill. So we found his address and gave him a visit. He wasn't there, but he left a sticky note on his slightly ajar front door which reads:

If you want my treasure, you must know how to swim.
How else would you dive to the bottom of the ocean?

He did the Home Alone thing, blocking all the drains and turning on all the faucets. We did find his computer, fully submerged in his running bathtub with another sticky note that said 'My Vessel'. Water got into the hard drive, but we're working to recover what we can.  

We did manage to recover this file pretty easily though. It was in a folder called "flag transfers". Go ahead and take a crack at it.
